# Codealpha_tasks

This repository contains the projects completed as part of the **CodeAlpha Data Science Internship**. Each project focuses on solving real-world problems using Python, covering data analysis, machine learning, and data visualization.

## Projects

### Task 1: Sales Prediction using Python

A machine learning project that predicts future sales based on advertising expenditure. The project includes data preprocessing, feature engineering, exploratory data analysis, Linear Regression model training, performance evaluation, and future sales prediction.

**Key Topics**

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Linear Regression
* Model Evaluation
* Future Prediction

---

### Task 2: Car Price Prediction using Machine Learning

A regression-based machine learning project that predicts the selling price of used cars using vehicle-related features. The workflow includes data preprocessing, visualization, feature engineering, model training, evaluation, and price prediction.

**Key Topics**

* Data Cleaning
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Linear Regression
* Model Evaluation
* Price Prediction

---

### Task 3: Unemployment Analysis with Python

A data analysis project that explores unemployment trends in India using Python. The project analyzes unemployment patterns across different regions, studies the impact of Covid-19, and presents insights using data visualization.

**Key Topics**

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Covid-19 Impact Analysis
* Trend Analysis
* Policy Insights
